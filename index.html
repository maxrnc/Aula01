HTML CSS Exemplos

O que é HTML?

HTML é a linguagem de marcação utilizada para estruturar o conteúdo da web.

O que é Css?

CSS é a linguagem utilizada para estilizar o HTML e deixar as páginas mais bonitas.

Exemplos de código

Exemplo de um título em HTML:

<h1> Olá, mundo </h1>

Exemplo de estilização com CSS:

h1 {color: blue}
